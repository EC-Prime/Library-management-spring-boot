# Library-management-spring-boot
A RESTful Library Management System built with Java and Spring Boot.

Description:
    Allows librarians and members to manage books and borrowing activities  through a REST API. 
    Built as part of a research-based assignment.

Tech Stack:
    Spring Boot
    Java 25
    H2
    Lombok
    Maven
    Spring Data JPA
    Hibernate
    MySQL
